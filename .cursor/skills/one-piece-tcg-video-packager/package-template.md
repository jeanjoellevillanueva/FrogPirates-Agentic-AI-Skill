## Package [A | B | C] — [Angle name]

### Title block

| Field | Value |
|-------|--------|
| **Full title** | (≤ ~70 chars ideal; list count in title) |
| **Headline** | 2–4 words ON thumbnail (largest text) |
| **Sub-headline** | Supporting line under headline |
| **Badge** | Corner label (INVESTOR GUIDE, PRICE ALERT, etc.) |

### Thumbnail block

| Field | Value |
|-------|--------|
| **Layout** | e.g. face left, card art right, number center |
| **Primary visual** | Card / box / chart element |
| **Text on thumb** | Headline + optional number (`+39%`, `TOP 7`) |
| **Color mood** | Green = money up / Red = warning / Gold = premium |
| **Avoid** | Clutter, unreadable text, mismatch with title |
