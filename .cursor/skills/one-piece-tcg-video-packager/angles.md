# Package emotion angles

Use **one angle per package**. Same topic, same list — different packaging.

## Angle A — FOMO / opportunity

**Triggers:** buy now, before, exploding, still early, won't stay cheap, last call

**Title patterns:**
- `…Before [Event]`
- `…Won't Stay Cheap`
- `…Exploded (+X%)`
- `MUST BUY … 2026`

**Headlines:** `BUY NOW`, `+39%`, `LAST CALL`, `MUST BUY`

**Badges:** `OPPORTUNITY WINDOW`, `HIDDEN GEMS`

**Hook tone:** Window closing; data just moved; stick around for ranked order.

---

## Angle B — Truth / authority

**Triggers:** ranked list, data, truth nobody tells, investor guide, here's the order

**Title patterns:**
- `TOP N … (Here's The Data)`
- `The Truth About …`
- `Ranked: Best … 2026`
- `…Explained`

**Headlines:** `TOP 7`, `RANKED`, `THE DATA`, `INVESTOR GUIDE`

**Badges:** `INVESTOR GUIDE`, `DEEP DIVE`, `SMART MONEY`

**Hook tone:** Calm expert; "let me break down the ranked list"; less scream, more credibility.

---

## Angle C — Fear / urgency

**Triggers:** wrong buy, don't sleep, losing value, too late, regret, crash

**Title patterns:**
- `Don't Buy … Blind`
- `…Or Regret It`
- `Stop Buying …`
- `…Is Getting Worse`

**Headlines:** `DON'T`, `WRONG BUY`, `TOO LATE`, `REGRET?`

**Badges:** `PRICE ALERT`, `URGENT`, `WARNING`

**Hook tone:** Validate panic → ranked solution; still deliver full list (no empty fear bait).

---

## JJ channel: which angle when

| Topic type | Lead with | Backup |
|------------|-----------|--------|
| Sealed spike week | A | B |
| Product pre-order stack | A or B | C |
| SP vs manga list | B | A |
| Bear market | C | B |
| Starter deck verdict | B or C | A |
| Budget under $X | A or E-style title | B |

Winner often **A or B** for finance; **C** only if data supports opportunity close.
