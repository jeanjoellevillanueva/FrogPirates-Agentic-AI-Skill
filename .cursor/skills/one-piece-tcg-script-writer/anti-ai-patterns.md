# Anti-AI patterns — script voice (JJ channel)

**Purpose:** Block phrases and structures that read as AI / LinkedIn / “mean girls hot take” energy.  
**Revert:** Delete this file and remove links from `SKILL.md` + `voice.md` to roll back.

**Use with:** [voice.md](voice.md) (cadence + tone). This file is the **phrase and sass** layer.

---

## Self-check (read script aloud)

- [ ] No banned openers or “truth bomb” pivots below
- [ ] No more than **one** devil’s advocate block per video (`Are you insane JJ?` → `Hear me out`)
- [ ] No stacked **X is not the killer / but actually Y** loops (max one pivot, then move on)
- [ ] No “Here's…” sentence starters more than **once** per script
- [ ] No `No [X]. No [Y]. Just [Z].` triplet closers
- [ ] Contrarian point uses **your** story (launch week tax, reprint headline, shop night) — not Wikipedia analogies piled up
- [ ] Sounds like JJ at the card shop, not a trailer or LinkedIn post

---

## 1. Forced sass / fake conflict

AI ramps “edge” with manufactured drama. JJ is direct, not performatively spicy.

| Banned (spoken script) | Use instead |
|------------------------|-------------|
| `But here's the thing:` | `But` + continue the sentence |
| `But here's the truth:` | Drop it — state the point |
| `But here's what nobody's saying:` | `I don't hear people talk about…` |
| `Hot take:` (in voiceover) | Title/thumbnail only — on camera say `Today we're going to talk about…` + the claim |
| `JJ from Frogpirates here` / `I'm JJ` (opening) | `What's up, pirates! Today we're going to talk about…` — see [opening-gate.md](opening-gate.md) |
| `Then I realized:` | `I figured out…` / `It clicked when…` |
| `The result?` | Say the result in a normal sentence |
| `And honestly?` | Cut it or `I'll be upfront` (once per video max) |
| `Let that sink in.` | Cut |
| `Read that again.` | Cut |
| `Full stop.` / `Period.` | Cut |

---

## 2. Cliché scene-setting openers

| Banned | Use instead |
|--------|-------------|
| `In today's fast-paced digital landscape…` | Name the actual thing: `TCGPlayer`, `OP-17 preorder`, `June reprint headline` |
| `In the dynamic world of…` | Cut |
| `As the world continues to evolve…` | Cut |
| `In an era of…` | Cut |
| `Shouting into the void` | Cut |
| `No fluff.` (as a spoken line) | Never say this on camera — skill rule “skip filler” is for writers, not VO |

---

## 3. LinkedIn / thesis-paper structures

| Banned pattern | Why | Fix |
|----------------|-----|-----|
| `No [X]. No [Y]. Just [Z].` | AI slogan closer | One sentence: what you actually do |
| `It is not just [X]. It's also [Y].` | Diplomatic AI balance | Pick the point that matters for the viewer’s wallet |
| `Here's why that matters:` | Lecture pivot | `So for your binder…` / `That means if you…` |
| `Here's the deal:` | Podcast AI | `The part that affects you:` |
| `Here's what I've started to believe:` | Fake journey | `I think…` / `After [specific event], I…` |
| `That's the real [unlock/challenge/killer].` | AI mic drop | Name the thing plainly once |
| `The real [X] is [Y].` (repeated) | Stacked reveals | Use once max per video, if at all |

---

## 4. Debate-bot loops (common in myth-bust scripts)

**Banned structure when repeated:**

```
[Thing] is not the killer.
[Paragraph of comparison.]
But / However / Yet…
[Opposite claim.]
```

**Allowed:** One clear table-clearing pass, then **story** (Japan, your shop, your bad buy) — not four identical essays.

**Reference scripts (human JJ):**  
`scripts/2026-06-never-buy-launch-week-script.md` · `scripts/2026-06-strategic-rebalancing-script.md` · `scripts/2026-06-ten-year-plan-survive-bubble-script.md`

---

## 5. Over-diplomatic / hedge stacking

| Banned habit | Use instead |
|--------------|-------------|
| `It's important to understand both sides…` | Take a side; one sentence of fair acknowledgment max |
| `This comes with advantages and disadvantages…` | Name the one disadvantage that costs **money** |
| `Looming challenges ahead…` | Name the challenge: `fewer locals`, `harder exit`, `reprint dip` |
| `It's worth noting that…` | Note it in plain language or cut |
| `At the end of the day…` | Cut |

## 7. JJ replacements (channel-specific)

| Instead of AI… | JJ pattern |
|----------------|------------|
| Hot take opener | `Wrong question` / `Today we're talking about why…` (Ten Year Plan) |
| Myth-bust stack | `Let me clear the table first` → one pass → **your** scene |
| Authority from comparisons | **Prices + dates**: `OP-16 box around two ten`, `June twelfth release` |
| Viewer connection | `If you're putting one to two hundred a month…` / `I live in those comments too` |
| Tension | One `Are you insane JJ?` → `Hear me out` per video |
| Close | Comment asks **their** mistake / country / BUILD or PROTECT — not quiz recap |

## Packaging vs voiceover

**OK in title, thumbnail, description, badge:** `HOT TAKE`, `6 TIPS`, `NOT REPRINTS`, `WRONG WAR` — CTR packaging.

**Not OK read aloud** unless it's how you'd literally say it to a friend without cringing.
