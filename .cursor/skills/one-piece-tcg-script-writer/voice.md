# JJ voice & investor rules

## Opening lines (rotate)

**Mandatory shape** — full gate: [opening-gate.md](opening-gate.md). Do **not** add channel name or `JJ from Frogpirates here` on camera.

- `What's up, pirates! Today we're going to talk about…`
- `What's up pirates! Today we're talking about why…`
- `WHAT'S UP PIRATES! Today, we're diving deep into…`

**Never as hook:** `JJ from Frogpirates here` · `Welcome back` · `In today's video` · spoken `Hot take:`

## Transition lines

- `Let's get into it.`
- `Let me break it down.`
- `Hear me out.`
- `I'll be upfront with you —`
- `And because you stayed until the end…`

## Tone

- Confident investor-educator, not hype bro
- Use plain calls: `screaming deal`, `must-buy`, `that's your signal`, `window is still open`
- Admit uncertainty when honest: `I believe`, `I genuinely think`, `the only question is`
- **Full sentences in first person** — talk like JJ on camera, not like a trailer voiceover
- One idea per beat; connect beats with *because*, *so*, *that's why*, or a concrete example

## Human voice — avoid AI cadence

Scripts must sound **spoken**, not **generated**. If a block reads like a LinkedIn post or a dramatic movie trailer, rewrite it.

### Banned patterns

| Pattern | Why it fails | Bad example |
|--------|----------------|-------------|
| **Fragment triplets** | Three punchy fragments in a row — no subject, no story | `The timeline freaks out. Prices wobble. Everyone acts like the set is dead.` |
| **Noun-stack fragments** | Telegram-style labels instead of sentences | `Side income. Learning the game. Portfolio growth.` |
| **Dramatic escalation stacks** | Fake tension built from parallel short lines | `Fear hits. Sellers panic. Smart money moves.` |
| **Binary label closers** | Neat slogan that sounds written, not said | `Noise for flippers and opportunity for planners.` |
| **Abstract crowd + mood** | `Everyone`, `the market`, `the timeline` with no specific scene | `Everyone's dumping. The market's bleeding. You're either in or out.` |

### Rewrite toward JJ

1. **Lead with I / you / we** — who's doing what?
2. **Name the real thing** — OP-13 reprint headline, TCGPlayer dip, Discord meltdown, not `the timeline`
3. **One cause → one reaction** — why does that moment matter for *your* plan?
4. **Read aloud** — if you wouldn't say it to a friend at the card shop, cut it

| AI cadence (cut) | Human JJ (use) |
|------------------|----------------|
| `The timeline freaks out. Prices wobble. Everyone acts like the set is dead.` | `OP-13 and EB-03 printing again in June twenty twenty-six — I've already seen people on Twitter acting like the set's finished. Prices dip. That's usually when I start looking, not when I panic sell.` |
| `Fear hits. Sellers panic. Smart money moves.` | `When Bandai confirms a reprint, weak hands sell first. I wait for the dip on stuff I already had on my list.` |
| `Side income. Learning the game.` | `I'm not trying to quit my job off one box — I'm learning which sets actually hold while I build a little side income.` |

### Still allowed (not the same as fragment spam)

- **Instructional triplets** when teaching a rule: `Let the opening volume hit. Let the rumors flush weak hands. Let the floor form.` — each line advances the same how-to.
- **List headers** in ranked videos: `#3 — OP04 SP Boa — $767`
- **One** short punch line for emphasis after a full explanation — not three in a row

### Sass, “Here's…”, and debate loops

Banned phrases, myth-bust stacking, and LinkedIn hot-take energy: [anti-ai-patterns.md](anti-ai-patterns.md). Run grep self-check on voiceover before handoff.


Use once per video max.

## Investor rules (state in script when relevant)

| Rule | Wording pattern |
|------|-----------------|
| English focus | `ONLY buy English cards` / `NOTE: I am talking about English sets` |
| Rarity floor | `SP and above` for long-term holds |
| Characters | `franchise pillars` — Luffy, Nami, Zoro, Boa |
| Reprint (EN) | Bandai typically reprints **once**; OP01 exception |
| Reprint (JP) | `For Japanese sets this is not applicable` |
| Precedent | Pokémon product line comparison when explaining sealed/SPC |
| Personal stake | `I pre-ordered a case` / `I broke even but didn't exit` |

## Thesis pillars (reuse across formats)

1. **Character popularity** — Straw Hats > film-only ceiling
2. **Community nickname** — Pika-Nami, Boa-Lisa = memory = price
3. **Iconic art** — frame-worthy, arc-defining
4. **Structural rarity** — pull math, gold leader slot, SP slot sharing, case odds
5. **Reprint damage** — manga hit harder than SP sometimes
6. **Out of print** — OP03 no reprints, set rotation
7. **Macro rotation** — Pokémon profits → One Piece landing spot
8. **Buy window** — not bullish = accumulate; post-anniversary cooldown

## Closings

```
That's it for today, pirates. If you found this helpful, hit that subscribe button.
I'll catch you in the next one. See you on our next voyage.
```

Variant comment CTA:

```
Comment down below: [specific question]
```

## Do not

- Write without numbers on ranked items
- Use wall-of-text — break every rank into its own block
- Copy competitor scripts line-for-line
- Invent prices — estimate with `~` and flag verification
