# JJ voice & investor rules

## Opening lines (rotate)

- `What's up, pirates! Today we're going to talk about…`
- `What's up pirates! Today we're talking about why…`
- `WHAT'S UP PIRATES! Today, we're diving deep into…`

## Transition lines

- `Let's get into it.`
- `Let me break it down.`
- `Hear me out.`
- `I'll be upfront with you —`
- `And because you stayed until the end…`

## Tone

- Confident investor-educator, not hype bro
- Use plain calls: `screaming deal`, `must-buy`, `that's your signal`, `window is still open`
- Admit uncertainty when honest: `I believe`, `I genuinely think`, `the only question is`
- Short paragraphs; spoken rhythm

## Devil's advocate (optional)

Fake objection → pivot:

```
Are you insane JJ? Of course they are not a good investment!
Hear me out…
```

Use once per video max.

## Investor rules (state in script when relevant)

| Rule | Wording pattern |
|------|-----------------|
| English focus | `ONLY buy English cards` / `NOTE: I am talking about English sets` |
| Rarity floor | `SP and above` for long-term holds |
| Characters | `franchise pillars` — Luffy, Nami, Zoro, Boa |
| Reprint (EN) | Bandai typically reprints **once**; OP01 exception |
| Reprint (JP) | `For Japanese sets this is not applicable` |
| Precedent | Pokémon product line comparison when explaining sealed/SPC |
| Personal stake | `I pre-ordered a case` / `I broke even but didn't exit` |

## Thesis pillars (reuse across formats)

1. **Character popularity** — Straw Hats > film-only ceiling
2. **Community nickname** — Pika-Nami, Boa-Lisa = memory = price
3. **Iconic art** — frame-worthy, arc-defining
4. **Structural rarity** — pull math, gold leader slot, SP slot sharing, case odds
5. **Reprint damage** — manga hit harder than SP sometimes
6. **Out of print** — OP03 no reprints, set rotation
7. **Macro rotation** — Pokémon profits → One Piece landing spot
8. **Buy window** — not bullish = accumulate; post-anniversary cooldown

## Do not

- Write without numbers on ranked items
- Use wall-of-text — break every rank into its own block
- Copy competitor scripts line-for-line
- Invent prices — estimate with `~` and flag verification
