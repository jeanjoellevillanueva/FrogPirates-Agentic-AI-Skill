# Golden script patterns (JJ)

These are the **canonical formats** for this channel. When writing new scripts, mirror structure and density — not word-for-word copying.

## Sample 1 — SP vs Manga (Format A)

**Title family:** `SP Cards That Matter More Than Manga Cards` / `…more expensive than manga?`

**Hook pattern:** Don't understand why it's a big deal → stick around → see why SPs are **quietly** becoming chase.

**Item pattern:**

```
#N — [Set] [Name] — $SP | [Manga/other]: $Y
[Flipped / closing gap / not yet but on list]
[Drivers: PRB01 reprint hit manga, character, art, OOP, nickname]
[5-year thesis one line]
```

**Framework close:** How to Spot — nickname, character popularity, iconic art, structural rarity (checklist).

## Sample 2 — Last call OP13/EB03 (Format B)

**Title:** `Last call: Buy OP13 and EB03 now or regret later`

**Point titles:** Reprint controlled (lottery 1000 qty) → deep value pool ($8k Luffy, $1.4k Uta…) → market cooled = buy window → Bandai rarely reprints twice (EN) → Pokémon money rotates.

**Close:** Stars aligned recap; `don't need all five`; time in market beats timing.

---

## Sample 3 — Starter decks (Format C)

**Title A/B/C packages** with WORTH IT / ONLY 4 MATTER / 4 HIDDEN GEMS badges.

**Arc:** Question → Pokémon comparator (weak ROI) → Bandai twist (RNG promos) → 4 ranked STs with promo math → verdict: not for ROI, yes for collectors.

---

## Sample 4 — Serialized (Format D)

**Title angles:** Hidden gem / Head to head / Wrong comparison.

**Arc:** Myth bust (not like manga supply) → lucky numbers table → numbered catalog (anniversary, tournament, tin) → advantages/disadvantages → bottom line.

---

## Sample 5 — Under $25 (Format E)

**Title variants:** BUY NOW, STOP OVERPAYING, $200 STRATEGY, MARKET SALE.

**Arc:** Market expensive → usually SP+ but today under $25 → 10 cards with price + catalyst each (live action, meta, Peach Momoko, rotation, CN exclusive).

---

## Sample 6 — Bear market (Format F)

**Title variants:** BOUGHT THE TOP?, MACRO CRASH, GETTING WORSE.

**Arc:** Validate panic → 5 reasons (scalpers leave, post-anniversary, macro, reprints, Logan top signal) → reframe cycle → EN/SP+/character rules → exit comment CTA.

## When user provides a topic from researcher

1. Map topic → format A–F (see script-writer `formats.md`)
2. Propose list size (enough real items with prices; don't pad)
3. Output title A/B/C + full script in one response unless user asks outline-only
