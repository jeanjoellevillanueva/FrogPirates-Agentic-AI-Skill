# Title packages (A / B / C)

Always deliver **three** variants. Each needs four fields for thumbnail team.

## Fields

| Field | Role | Examples |
|-------|------|----------|
| **Title** | Full YouTube title | `10 SP Cards That Matter More Than Manga Cards` |
| **Headline** | 2–4 words on thumbnail | `HIDDEN GEMS`, `BUY NOW`, `WRONG` |
| **Sub-headline** | Supporting line | `The chase cards nobody talks about` |
| **Badge** | Corner label | `INVESTOR GUIDE`, `DEEP DIVE`, `PRICE ALERT` |

## Angle matrix (pick one per variant)

| Variant | Emotion | Title pattern |
|---------|---------|---------------|
| **A** | FOMO / opportunity | `Buy Now`, `Before It's Too Late`, `Will Be Worth 10x` |
| **B** | Contrarian / truth | `The Truth Nobody Tells You`, `Stop Comparing X to Y` |
| **C** | Fear / urgency | `Watch This Now`, `Losing Value`, `Don't Sleep On` |

## List count in title

Title count **must match** script count:

- `10 MUST have cards` → 10 entries
- `4 Starter Decks` → 4 entries
- `8 Cards Under $25` → 8 entries

## Thumbnail pairing

- Headline = largest text
- Badge = corner contrast color
- Sub-headline = under headline, smaller
- Visual = #1 card/product or split comparison (SP vs Manga)
