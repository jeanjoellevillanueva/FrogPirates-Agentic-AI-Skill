# Script formats (JJ channel)

Pick **one** format per video. Every format ends with a list; only the **label style** changes (`#N` vs `Point N`).

---

## Format A — Ranked chase list

**Examples:** SP vs Manga, serialized tiers, top cards under $25, SPs that flipped manga.

**Structure:**

```
Hook → "Let's get into it." →
#1 — [Card] — $X | vs [benchmark]: $Y
  [Drivers: reprint, nickname, character, pull rate, trend]
#2 — ...
...
Optional: "How to Spot the Next One" (3–4 signals)
Outro
```

**Per-item formula:**

1. Rank + set + nickname if any
2. Price line (both sides of comparison)
3. Status: flipped / closing gap / hasn't flipped yet (why still on list)
4. 2–4 proof points (PRB reprint hit manga, out of print, community name)
5. Forward thesis (`5 years`, `when gap closes`)

**List size:** 8–10 for card rankings; 5–7 for products.

---

## Format B — Last call / thesis points

**Examples:** Buy OP13 + EB03 now; must-buy pre-orders stack.

**Structure:**

```
Hook → "Let me break it down." →
Point 1 — [Bold claim]
  [Evidence: distributor behavior, lottery caps, prices]
Point 2 — ...
...
Optional bonus: "Because you stayed until the end…" (2 extra picks + tease other video)
Closing stack: "You don't need all five…" + stars aligned recap
Outro
```

**Per-point formula:**

1. Named point (reprint controlled, deep value pool, buy window, etc.)
2. Counter the objection (`I know what you're thinking…`)
3. Specific numbers ($8k Luffy manga, five cards over $1k)
4. English-only rule when relevant

**List size:** 4–6 points + 0–2 bonus items.

---

## Format C — Few winners + verdict

**Examples:** Starter decks worth buying (4 of 29).

**Structure:**

```
Hook → question in title →
Optional: comparator frame (Pokémon battle deck vs league deck — conclude ROI weak)
Devil's advocate exchange
"If you want to know which, stick around."
#1 — [Product ID + name]
  [Promo RNG, variations count, chase card price, peak vs now]
#2 — ...
Final verdict: "Not really" / "Only if collecting" / stick to boxes
CTA question for comments
Outro
```

**List size:** 3–5 winners; explicitly state **denominator** (e.g. "4 out of 29 decks").

---

## Format D — Deep dive catalog

**Examples:** Serialized cards explained; lucky numbers.

**Structure:**

```
Hook → "you've got it completely wrong" (myth bust) →
Explain mechanic (manga vs serialized supply logic)
Teach framework (good numbers / bad numbers) — bulleted
Numbered catalog of real assets (1–10+)
WHY IT MATTERS — Advantages / Disadvantages bullets
Bottom line + comment question
Outro
```

**List size:** 8–12 catalog entries; framework before or after catalog.

---

## Format E — Budget ranked

**Examples:** 10 must-have cards under $25.

**Structure:**

```
Hook → acknowledge expensive market →
"I always advise SP and above BUT today under $X" →
#1 — [Card] — $XX.XX
  [Why: first of kind, meta staple, promo capped, artist, EN exclusive]
...
Outro
```

**Rules:**

- Every item **≤ cap** (or explain one exception)
- Mix: promos, meta staples, artist cards, exclusives
- Tie to **future catalyst** (show, rotation, no EN equivalent)

**List size:** 8–10.

---

## Format F — Bear market / fear list

**Examples:** Why cards are losing value; is OP TCG done?

**Structure:**

```
Hook → validate panic → promise real reasons →
1. [Reason] — [mechanism] — [what pros do vs scalpers]
2. ...
...
Reframe: predictable cycle, not death
Rules to survive (EN only, SP+, popular characters)
Optional: influencer top signal (name only if relevant)
Comment question: "are you going to exit?"
Outro
```

**List size:** 5–6 reasons; emotional but end on opportunity.

## Cross-format add-ons

| Add-on | When |
|--------|------|
| **Devil's advocate** | Formats B, C — 1 exchange before list |
| **Pro tip / framework** | Format A, D, E — after list |
| **End bonus** | Format B — 2 extra picks + video tease |
| **Comment CTA** | All — one specific question |
| **Pokémon comparator** | Format C, B (macro rotation) |
