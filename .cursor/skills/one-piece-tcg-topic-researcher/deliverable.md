# Final deliverable template

Use this structure for every research run. Replace placeholders with real numbers from the session.

---

## 1. Best topic

**[One-line topic name]**

Example: *Blue Purple Luffy Is Warping OP11*

---

## 2. Why it is trending

| Signal | Value |
|--------|--------|
| Creators covering (7d) | N |
| Avg performance ratio | X.XX |
| Reddit hype score | 0–5 |
| Meta / tournament relevance | 1–2 sentences |

Bullet supporting evidence (3–5 bullets max):

- Creator overlap examples (channel + ratio highlight)
- Standout small-creator viral (if any)
- Meta site / tournament tie-in
- Reddit thread themes (frustration vs hype)

---

## 3. Script format & list plan

| Field | Value |
|-------|--------|
| **Format (A–F)** | A=ranked chase, B=last call, C=few winners, D=deep dive, E=budget, F=bear list |
| **List size** | e.g. Top 10 cards, 5 thesis points, 4 of 29 products |
| **Rankable items (draft)** | Bullet each #1–#N with one-line angle + price if known |

Must be filmable as a **numbered list**. If fewer than 5 items, downgrade topic or merge cluster.

**Next step:** Run [one-piece-tcg-video-packager](../one-piece-tcg-video-packager/SKILL.md) with this section — do not write full Title A/B/C here.

---

## 4. Packaging seeds (for video-packager)

Provide **3 angle seeds** only (not full packages):

| Seed | Emotion | Working title idea | Thumb number/$ |
|------|---------|-------------------|----------------|
| A | FOMO | … | e.g. +39% |
| B | Authority | … | e.g. TOP 7 |
| C | Fear | … | e.g. DON'T |

Packager turns seeds into coherent packages. Script writer uses **winning package** only.

Favor emotional curiosity:

- Fear / problem: *"OP11 Has A Serious Problem..."*
- Panic / quit: *"Why Everyone Is Quitting One Piece TCG"*
- Broken / ban: *"Bandai Needs To Ban This"*
- Money: *"This Card Just Spiked 52%"*

---

## 5. Thumbnail angle

One primary concept + 2 alternates. Examples:

- Shocked face + giant green ROI number
- Red warning symbols + *"BROKEN?"*
- Split card art vs price chart arrow

---

## 6. Unique angle (do not copy)

Include a **script outline** as numbered list items (#1… or Point 1…), not vague bullets:

- Format A: each rank = price line + vs benchmark + 2 drivers
- Format B: each point = objection + evidence + number
- Format C: denominator + verdict
- See [script-writer formats.md](../one-piece-tcg-script-writer/formats.md)

Explicitly state what **existing** creators are doing, then your **different** frame:

- Market impact (western buyout, sealed, ROI)
- Hidden tech / budget substitute
- Counter-meta or "everyone missed this"
- Emotional framing competitors skipped

---

## 7. Timing & saturation

| | |
|--|--|
| Saturation risk | Low / Medium / High |
| Publish urgency | Film within 24–48h / This week / Can wait |
| Pivot if late | One-sentence backup angle |

---

## 8. Runner-up topics (optional)

Top 2 alternates with scores and one-line angle each.
